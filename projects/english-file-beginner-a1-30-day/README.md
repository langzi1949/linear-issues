# English File Beginner A1 30-Day Plan

This project contains a 30-day English learning plan built for Linear import.

## Scope
- Main material: `English File Beginner`
- Supporting material: `English Pod`
- Duration: `30 days`
- Daily study time: `5 hours`
- Goal: finish `English File Beginner` in 30 days and build practical `A1` ability

## Files
- `linear_issues.csv`: Linear import file with 60 daily issues

## Task Design
- `2 days` per `File`
- After every `2 Files`, insert `Practical English + Revise & Check`
- Each study day is split into `2` issues:
  - one `English File` issue
  - one `English Pod` issue
- Each issue includes:
  - a single focus area
  - measurable output tasks
  - a same-day acceptance standard
