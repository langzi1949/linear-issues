# AI Agents LangGraph MCP RAG Milestones

This file defines the milestone titles and descriptions used by `linear_issues.csv`.

## Milestone 1
Title: Book Study Foundations
Description: Complete the first stage of book study, covering Chapters 5-8 and the first review cycle that consolidates the core AI Agent foundations for the demo project.

## Milestone 2
Title: Book Study Advanced Patterns
Description: Complete the second stage of book study, covering Chapters 9-12 and the second review cycle that consolidates advanced RAG, workflow, and evaluation patterns.

## Milestone 3
Title: RAG Demo Design
Description: Finish the last book-study stage and convert the learning into a Wanli Fifteen Years AI Agent RAG demo design, including scope, retrieval, and workflow planning.

## Milestone 4
Title: Final Synthesis and Output
Description: Close the project with a final synthesis of the book learning, the demo design, and the next-step implementation outputs.
